# Real Estate CRM

This is a full-stack CRM application for managing real estate clients and leads.

## Tech Stack
- Java
- Spring Boot
- React
- Database (MySQL/Postgres)

## Features
- Create, edit, delete clients
- Track client stages
- Filter by stage
- Inline editing

## How to Run

### Requirements
Make sure you have installed:
- Java 17+
- Maven
- Node.js
- PostgreSQL

---

### Backend
mvn spring-boot:run

### Frontend
npm install
npm run dev

## Database Setup

This project uses PostgreSQL.

Update the database credentials in:
src/main/resources/application.properties
